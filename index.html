Cuberto Mouse
Follower

We present a redesigned version of our library to create amazing and smooth mouse cursor effects on your site. Meet Cuberto Mouse Follower! Now this is a full-fledged and powerful JavaScript library that we have published in the NPM, which means that we will periodically improve and supplement it. The new version has support for creating a skewing effect, the ability to add images, videos and icons to your cursor and many improvements and features. Hurry up to try it!
